# Spice Route Restaurant Website

A visually rich, semantic, and multi-page website built from scratch for **Spice Route Restaurant**—an authentic culinary experience established in 1998 in New Delhi. This project showcases clean HTML5 structuring and modular CSS3 styling without the use of heavy external frameworks.

## 🚀 Live Demo
👉 [View Live Project]( https://mohdnafisalam.github.io/PROJECT-RESTAURANT/)

---

## ✨ Features

* **Root-Level Entry (`index.html`):** Built with the primary landing page in the root directory for seamless web server hosting and deployment (e.g., GitHub Pages).
* **Multi-Page Navigation:** Seamlessly browse between different menus and information pages across subdirectories.
* **Modular Stylesheets:** Uses dedicated CSS files (e.g., `style.css`, `abouts.css`, `drinks.css`) to keep page-specific styling organized and maintainable.
* **Semantic Structure:** Implements modern HTML5 structural tags (`<header>`, `<nav>`, `<main>`, `<section>`) for optimal accessibility and SEO.
* **Structured Data Layouts:** Showcases restaurant menu items dynamically using cleanly styled HTML tables complete with descriptions and currency symbols (₹).
* **Interactive Navigation:** Includes an active-state navigation bar so users always know which page they are currently browsing.

## 📂 Project Structure & Pages

The project is structured with the main landing page at the root level and supporting pages/assets organized neatly into subfolders:

```text
PROJECT 2/
├── CSS/
│   ├── abouts.css      # Custom styles for the About page
│   ├── drinks.css      # Custom styles for the Beverages & Desserts page

├── HTML/

│   └── drinks.html     # Beverages & Desserts page
└── index.html          # Starters & Mains page (Main entry point)
